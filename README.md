# Nritya-Quest: Indian Dance Forms Showcase

## Overview
Nritya-Quest is a web-based application designed to showcase Indian dance forms categorized by state. It provides an interactive platform for users to explore and learn about the rich and diverse dance traditions of India.

## Features
- **State-wise Categories**: Explore various Indian states and their unique dance forms.
- **Multimedia Integration**: Includes high-quality images and descriptions for each dance form, enhancing the visual experience.
- **Interactive Elements**: Navigate seamlessly through the application and interact with content dynamically.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Acknowledgements
- **List of Indian Classical Dances**: Used as a reference for categorizing dance forms by state.

## Usage
1. Clone the repository:
   ```
   git clone https://github.com/your_username/nritya-quest.git
   ```
   
2. Open `index.html` in your web browser to start exploring the application.
