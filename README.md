# Nritya-Quest
Indian Dance Forms Showcase
Overview
This project is a web-based application designed to showcase Indian dance forms categorized by state. It aims to provide an interactive platform for users to explore and learn about the diverse dance traditions of India.

Features
State-wise Categories: Explore various Indian states and their unique dance forms.
Multimedia Integration: Includes images and descriptions for each dance form.
Interactive Elements: Allows users to navigate seamlessly and interact with content dynamically.

Technologies Used
HTML5
CSS3
JavaScript
